# Writing Plans

## Plan Structure
1. **Goal** — what success looks like (measurable)
2. **Scope** — what's included and explicitly excluded
3. **Milestones** — ordered deliverables with dates
4. **Risks** — what could go wrong and mitigation
5. **Dependencies** — what must happen first

## Implementation Plan Template
```markdown
# [Feature Name]

## Objective
[One sentence: what we're building and why]

## Success Criteria
- [ ] Criterion 1 (measurable)
- [ ] Criterion 2

## Phases
### Phase 1: Foundation (Week 1)
- Task A
- Task B

### Phase 2: Core Logic (Week 2)
- Task C
- Task D

## Risks
| Risk | Probability | Impact | Mitigation |
|------|------------|--------|------------|

## Out of Scope
- [Explicitly list what this does NOT cover]
```
